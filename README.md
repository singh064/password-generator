# Password Generator 🔐

This is a simple Python project that generates random passwords using letters, numbers, and symbols. I made this as part of my internship practice to improve my Python basics.

## Features

* Generates random passwords
* You can choose:

  * number of letters
  * number of symbols
  * number of numbers
* Can generate multiple passwords at once
* Password is shuffled so it's not predictable

## How it works

The program takes input from the user and then randomly picks characters from predefined lists. After that, it shuffles them and joins everything into a final password.

## How to run

Just run the Python file:


python password_generator.py



## Example

1. aB3$kP9!
2. Z#8mL2@q
